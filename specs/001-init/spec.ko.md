# 명세서: 초기 프로젝트 설정 (001-init)

## 1. 개요
블로그 플랫폼을 위한 기초 인프라를 구축합니다. 간단한 도메인 모델을 사용하여 스키마 주도 개발(SDD) 워크플로우를 검증하는 것이 목표입니다.

## 2. 목표
- TypeScript 기반의 Node.js 환경 설정.
- Prisma v6 연동 및 동작 확인.
- 아키텍처 검증을 위한 기본적인 "사용자(User)" 및 "게시글(Post)" 관리 기능 구현.

## 3. 기능 요구사항
### 3.1 사용자 관리 (User Management)
- **사용자 생성**: 이메일과 이름(선택)으로 신규 사용자를 등록할 수 있어야 함.
- **사용자 조회**: 사용자 정보를 조회할 수 있어야 함.

### 3.2 게시글 관리 (Post Management)
- **게시글 작성**: 사용자는 제목과 내용(선택)을 입력하여 글을 작성할 수 있음.
- **게시글 발행**: 초안 상태의 글을 발행(Publish)할 수 있음.
- **피드 조회**: 발행된 게시글 목록을 조회할 수 있음.

## 4. 비기능 요구사항
- **타입 안전성**: 모든 입력과 출력은 Prisma가 생성한 타입을 사용해야 함.
- **데이터 무결성**: 데이터베이스 제약조건(이메일 중복 불가 등)을 준수해야 함.
